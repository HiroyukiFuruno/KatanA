# Sample Spec-Driven Workspace

This workspace is a fixture used for validating Katana's MVP features.

## Contents

- `docs/architecture.md` — architecture overview with a Mermaid diagram
- `docs/sequence.md`     — sequence diagram using PlantUML
- `docs/data-model.md`  — Draw.io embedded diagram
- `spec.md`             — top-level spec document

## About

This fixture is designed to exercise all supported diagram block types:
`mermaid`, `plantuml`, and `drawio` --- using only MVP-supported input formats.
