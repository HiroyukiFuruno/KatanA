# Project Spec

## Goal

Build a macOS desktop editor for spec-driven development with:

- Local workspace navigation
- Markdown authoring with GFM support
- Live preview with inline diagram rendering (Mermaid, PlantUML, Draw.io)

## Acceptance Criteria

- [ ] Workspace can be opened from a local directory
- [ ] Markdown files can be edited and saved
- [ ] Preview updates without requiring a file save
- [ ] Mermaid blocks render as inline diagrams
- [ ] PlantUML blocks render as inline diagrams
- [ ] Draw.io blocks render as inline diagrams
- [ ] Invalid diagram blocks show a fallback without crashing
- [ ] App is usable with no AI provider configured
