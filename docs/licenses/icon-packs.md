# Icon Packs & Licenses

KatanA bundles several high-quality, open-source icon packs to allow users to customize their working environment. All icon packs are distributed under their respective permissive open-source licenses as follows:

## 1. KatanA (Default)

- **Author**: Hiroyuki Furuno
- **Source**: <https://github.com/HiroyukiFuruno/KatanA>
- **License**: MIT License
- **Usage**: Core application icons bundled internally.

## 2. Material Symbols

- **Author**: Google
- **Source**: <https://fonts.google.com/icons>
- **License**: Apache License 2.0
- **Usage**: Outlined design variants used as selectable theme.
- **Modifications**: Re-mapped system/brand icons to fallbacks where applicable.

## 3. Lucide

- **Author**: Lucide Contributors
- **Source**: <https://lucide.dev/>
- **License**: ISC License
- **Usage**: Used as selectable icon theme pack.

## 4. Tabler Icons

- **Author**: Paweł Kuna / Tabler
- **Source**: <https://tabler.io/icons>
- **License**: MIT License
- **Usage**: Outline icon variants mapped to Katana namespaces.

## 5. Heroicons

- **Author**: Tailwind Labs Inc.
- **Source**: <https://heroicons.com/>
- **License**: MIT License
- **Usage**: 24px Outline variants mapped to Katana namespaces.

## 6. Feather Icons

- **Author**: Cole Bemis
- **Source**: <https://feathericons.com/>
- **License**: MIT License
- **Usage**: Primary icon theme selection and fallback mapping for some missing brand icons in other datasets.

---
*Note: Some brand icons (e.g. Github) or proprietary logos are not bundled in specific generic icon sets by their authors due to licensing or trademark restrictions. In such cases, KatanA utilizes highly-compatible alternatives from other permissive sets (such as Feather).*
