# OpenSpec

このディレクトリは、Katana の正式な変更提案と設計・タスク管理の正本。

## 置かれるもの

- `changes/<change-id>/proposal.md`: 変更提案
- `changes/<change-id>/design.md`: 設計
- `changes/<change-id>/tasks.md`: 実装タスク
