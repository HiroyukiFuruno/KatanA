# v0.22.2 Design: i18n, Scroller & Shortcut UX

## 1. i18n システムの補完
- `Close Document`, `Restore Closed Document`, `Global Search` 等の重要コマンドが多言語環境で英語表示されている問題を解消。
- 全 11 ロケールの `*.json` ファイルを走査し、未翻訳の値を補完する。

## 2. コマンドパレットスクロールの修正 (TDD)
- `egui::ScrollArea` の実装を見直し。特に `max_height` がウィンドウサイズを超えて設定されている、あるいは親コンテナのレイアウト制約によってスクロール領域が正しく計算されていない箇所を特定する。
- 物理的なスクロール挙動（マウスホイール、スクロールバーのドラッグ）が、大量の項目リストにおいて意図通り（1行ずつ、あるいは慣性を持って）動作することを保証する。

## 3. ショートカット設定 UI の再設計
- **録音モーダル**: ショートカットキーの入力を、行内のインプレース編集から専用のモーダル表示へ移行。録音中は `egui` の KeyFilter を使用してシステムショートカット（Cmd+W等）の誤動作を防止する。
- **ネイティブアイコン表示**: `⌘`, `⇧`, `⌥`, `⌃` 等の記号を SVG アイコンとして描画し、視認性を高める。
- **検索/フィルタ**: コマンド名だけでなく、現在割り当てられているキー（例: "Cmd+B"）での検索にも対応させる。
