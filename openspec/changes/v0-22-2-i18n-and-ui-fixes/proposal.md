# v0.22.2: i18n 改善およびコマンドパレットUI修正

## 1. 背景・目的
Katana v0.22.1 のリリースに伴い、一部のコマンド名が英語のまま残っている箇所（i18n 漏れ）が発見されました。また、コマンドパレットにおいて検索結果が多い場合に、マウスやキーボードでのスクロール挙動が不正（または使いづらい）という課題があります。
本変更では、ユーザー体験のさらなる向上のため、これらの不具合を TDD (RED) プロセスを遵守して修正します。また、v0.22.1 から劣後させたショートカット設定 UI の改善を統合します。

## 2. 主な変更点
- **i18n の完全補完**: 
  - コマンドパレットやショートカット設定画面において、"Close Document", "Restore Closed Document" 等が日本語環境でも英語で表示されている問題を修正。
  - 全 11 ロケールにおいて、対応する翻訳が不足している箇所を補完。
- **コマンドパレットのスクロール挙動修正**:
  - 検索結果が多い場合の縦スクロールが正しく機能しない、あるいは表示エリアを突き抜ける不具合を修正。
  - これを **TDD (RED)** で再現し、修正後にグリーンになることを確認する。
- **ショートカット設定 UI の改善 (v0.22.1 からの引継ぎ)**:
  - 録音モーダルの導入、OSネイティブキーアイコンの表示、検索バーの追加など、UX を大幅に改善。

## 3. 期待される成果
- 多言語環境において UI の一貫性が保たれること。
- コマンドパレットが大量のコマンドに対しても快適に操作可能になること。
- ショートカット設定が直感的かつ効率的に行えるようになること。
