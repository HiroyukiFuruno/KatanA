# Table of Contents Improvements (v0.22.6)

## Summary

This change improves the table of contents functionality by removing unnecessary backgrounds, adding accordion support, and enhancing navigation controls.

## Problem

The current table of contents lacks proper visual design and navigation capabilities, including absence of accordion functionality and improper background styling.

## Solution

1. Remove background styling from TOC headings
2. Implement accordion support for expanding/collapsing sections
3. Add full open/close controls in the TOC
4. Add vertical lines to accordion sections with themeable setting

## Acceptance Criteria

- TOC headings have no background styling
- Accordion functionality for expanding/collapsing sections
- Full open/close controls at the top of the TOC
- Vertical lines in accordion sections with themeable setting
