## このタブについて

このタブは **Demo バンドル** の一部です。**「Help → Demo」** から開くことができる、
KatanA の機能をアプリ内で体験するためのキュレーションされたファイルセットです。

このデモグループのファイル構成:

| ファイル | 目的 |
|------|---------|
| `welcome.ja.md` | この紹介文（Markdown、編集可） |
| `rendering_features.md` | 描画機能ショーケース |
| `katana_config.toml` | 設定ファイルの例（参照専用） |
| `sample_code.rs` | Rust コード例（参照専用） |

> [!TIP]
> このデモのコード・設定ファイルは **リファレンスモード** で開かれます — 読み取り専用のため、
> バンドルされたアセットを誤って変更することなく安全に閲覧できます。

## 主な機能

### ✍️ Markdown 編集

KatanA は GitHub Flavored Markdown をリアルタイムで描画します。このグループの
`rendering_features.md` を開くと、見出し、リスト、テーブル、引用、タスクリスト、数式などを確認できます。

### 🖼️ ダイアグラム対応

Mermaid・PlantUML・DrawIo のフェンスコードブロックを貼り付けるだけで、
プレビューペインに自動描画されます。ブラウザ不要です。

```mermaid
graph LR
    エディタ -->|描画| プレビュー
    プレビュー -->|クリックでジャンプ| エディタ
```

### 🗂️ タブグループ

このデモの全ファイルは **demo** タブグループに属しています。
任意のタブを右クリックすることで、独自のグループを作成できます。

### 🌐 多言語対応

KatanA は英語・日本語・中国語（簡体字・繁体字）・韓国語・ポルトガル語・
フランス語・ドイツ語・スペイン語・イタリア語に対応しています。
**KatanA → 言語** から切り替えられます。

## 独自ワークスペースを始める

1. **「ファイル → ワークスペースを開く」**（⌘O）でフォルダを開きます。
2. 左サイドバーからファイルをナビゲートします。
3. タブのレイアウトとスクロール位置はセッション間で自動的に保存されます。

---

## バンドル内のリファレンスファイル

設定ファイルフォーマットの実例は `katana_config.toml` を、
シンタックスハイライト付きコード例は `sample_code.rs` を開いてください。

どちらも **リファレンスモード** で開かれるため編集できません。
これによりデモバンドルの誤変更を防ぎます。
