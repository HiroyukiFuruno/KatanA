# Linter Fix Feature — 検証用ドキュメント

KatanA の Lint Fix 機能の動作確認に使うファイルです。
このファイルには意図的な lint 違反が含まれています。

---

## Fix 機能の使い方

1. エディタでこのファイルを開く（コードモード or 分割表示）
2. 行番号の左に表示される **💡 アイコン**にカーソルを当てる
3. ツールチップに **`Fix`**（この行のみ修正）と **`Fix All`**（同一ルール全修正）ボタンが表示される
4. クリックすると該当箇所が自動修正される

> **注意**: Fix ボタンはアイコンが表示される行にのみ出現します。
> アイコンが出ない行はその違反が fixable でないことを意味します。
